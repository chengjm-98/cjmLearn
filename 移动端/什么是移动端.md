# 什么是移动端

H5 / 小程序 / UniApp / React Native 都不是“移动端”的全部

- **“移动端开发” = 在手机上运行的应用的不同实现方式**

```jsx
手机上运行的应用
│
├─ 原生 App（Native）
│
├─ WebView 容器
│   └─ H5 / Hybrid
│
├─ 小程序
│
└─ 跨端框架
   ├─ React Native
   ├─ UniApp
   └─ Flutter
```

# 什么是 H5

- 网页，专门运行在手机浏览器或者 app 上的 webview 中运行的网页
  - 特点
    - 一套代码，多端运行
    - 可以使用浏览器的所有功能，但是不能使用手机的所有功能/api
- 所有小程序 / 跨端 / Hybrid，底层思想都来自 H5

# 什么是 hybrid

- app 里面嵌入一个 webview，然后在 webview 里面运行的网页

# 什么是小程序

- 被平台管控的 App
  - 特点
    - 不是网页，不是浏览器，有自己的一套运行环境
  - 技术特点
    - 类 Web 语法
    - 受平台限制
    - 不能直接用 DOM

## 本质

小程序 ≠ H5  
 小程序 ≠ 原生  
 它是 平台定制的“半原生”方案

# 什么是 UniApp

- 跨端框架，支持小程序、H5、App 等多种平台
- 写一套代码，编译到多端
- vue 语法，统一 api

# 什么是 React Native

- 用 React 写“真 App”

```jsx
React 代码
↓
Native UI（不是 WebView）
```

- 性能接近原生
- 真正的 App
- 可以写原生模块

# 路线

```jsx
入门            进阶                高级
H5 → 小程序 → Hybrid → RN / 原生
```
