# - 两个类，一个是地图类，一个是点位类normalmarker

- 封装一个地图类继承mapboxgl.Map
- 这个类里面包括constructor构造函数、initMap初始化地图、addMarker添加标注、addPopup添加弹窗/flyto等方法/addlayer添加图层/markerlist
- 关于不同的点位类型，封装了一个点位的类，包括点的dom元素，经纬度，名称，点击方法等
- 然后如果每个点位的dom如果想要自定义，可以传入jsx元素，然后通过createElement方法创建dom元素，然后将dom元素传入点位类的构造函数中。还支持zoom动态放大缩小
- 然后在需要打点的地方，先自定义生成点位的实例，然后调用地图类的addMarker方法，将点位实例传入，即可实现打点功能。

# 对于非正常的点位，比如需要聚合和自己特殊逻辑的点位，可以重新新建一个specialMarker类。

# 那么如果比如点位类型多的情况下，可以扩展一下创建一个点位的工厂函数，然后根据type使用不同的策略类，比如普通点位，特殊点位，聚合点位等。
