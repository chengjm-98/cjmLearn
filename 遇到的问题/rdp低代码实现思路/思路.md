# 数据结构设计

- 低代码的核心是组件树（Component Tree）。每个节点代表一个组件，有属性、事件和子节点。

```jsx
interface ComponentNode {
  id: string; // 唯一ID
  type: string; // 组件类型，例如 Button / Input / Container
  props: { [key: string]: any }; // 属性，如文本、颜色、尺寸
  events?: { [key: string]: string }; // 事件绑定，如 onClick: "submitForm()"
  children?: ComponentNode[]; // 子节点（嵌套组件）
  layout?: { x: number, y: number, width: number, height: number }; // 画布布局信息
}
```

# 拖拽实现

- 使用 React + React-DnD / interact.js / plain mouse events 实现拖拽。
- 拖拽逻辑：
  - 拖拽开始：生成一个新的 ComponentNode，放到画布组件树里。
  - 拖拽移动：更新 layout.x、layout.y。
  - 拖拽结束：固定位置。

# 画布渲染

- 递归渲染组件树
- elayout 控制绝对位置，props 控制样式和内容

# 配置管理

- 配置面板：每个组件都有配置面板，用户可以编辑属性、事件等。
- 可以实时修改属性
- 可以扩展事件绑定和数据绑定表达式
