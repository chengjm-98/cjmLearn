React Scheduler 是一套自研的任务调度器，通过链表队列管理任务，根据 lane 优先级映射到 Scheduler Priority。它使用 scheduleCallback 注册任务，浏览器事件循环驱动 workLoop 执行 Fiber 渲染，并通过 shouldYield 实现时间切片和可中断渲染，高优先任务可打断低优先任务，commit 阶段不可中断。Scheduler 内部兼容多种浏览器 API（MessageChannel 等）实现并发渲染。

## - 为什么弃用的原因

      - 不稳定，不同的浏览器实现不一致问题，尤其是 Safari
      - 粒度粗，只能在浏览器认为空闲的时候执行，不能细粒度控制
      - 受限于帧调度
