# OSI 七层模型

- 应用层：应用程序直接使用的接口，如 HTTP、FTP、SMTP 等。
- 表示层：数据的表示、安全、压缩等处理。
- 会话层：建立、管理和终止会话。
- 传输层：提供端到端的可靠数据传输，如 TCP、UDP 等。
- 网络层：路由选择、流量控制等网络管理。
- 数据链路层：帧的封装、解封装、错误检测等。
- 物理层：比特流的传输，如网线、光纤等物理媒体。

# TCP/IP 四层模型。

- 网络接口层（Network Interface Layer）：包含物理层和数据链路层的功能，负责数据在本地网络中的传输。
- 互联网层（Internet Layer）：主要包括 IP 协议，负责地址分配和路由。
- 传输层（Transport Layer）：提供端到端的可靠传输，常用的协议是 TCP 和 UDP。
- 应用层（Application Layer）：为用户提供各种网络服务，像 HTTP、FTP、SMTP 等协议。
