# 使用标准 CSS 属性 + 前缀（Vendor Prefix）

| 前缀       | 浏览器                    |
| ---------- | ------------------------- |
| `-webkit-` | Chrome, Safari, 新版 Edge |
| `-moz-`    | Firefox                   |
| `-ms-`     | IE, Edge（旧版）          |
| `-o-`      | Opera（老版）             |

- 可以引入 Autoprefixer 插件，让浏览器自动添加前缀。
