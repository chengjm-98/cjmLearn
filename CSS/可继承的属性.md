# css 属性继承

CSS 的继承指 子元素默认会沿用父元素的某些样式。
并非所有 CSS 属性都会继承，只有部分属性是“自然继承”的。

# 可继承属性

- 文本相关属性（自然继承）
  | 属性 | 说明 |
  | ---------------- | --------------- |
  | `color` | 文字颜色 |
  | `font-family` | 字体 |
  | `font-size` | 字体大小 |
  | `font-style` | 字体样式（italic） |
  | `font-variant` | 字体变体 |
  | `font-weight` | 粗细 |
  | `line-height` | 行高 |
  | `letter-spacing` | 字母间距 |
  | `word-spacing` | 单词间距 |
  | `text-align` | 文本对齐方式（部分浏览器继承） |
  | `text-transform` | 大小写转换 |
  | `visibility` | 可见性（继承但有些特殊情况） |
  | `white-space` | 空白处理 |
- 其他继承属性
- cursor（光标类型）
- quotes（引号样式）
- 列表属性
  | 属性 | 说明 |
  | --------------------- | ------ |
  | `list-style-type` | 列表标记类型 |
  | `list-style-image` | 列表图标 |
  | `list-style-position` | 标记位置 |

# 不可继承属性

- 大多数 盒模型相关属性、布局和边框、背景都 不继承，例如：
  | 属性 | 说明 |
  | --------------- | ---------- |
  | `margin` | 外边距 |
  | `padding` | 内边距 |
  | `border` | 边框宽度/颜色/样式 |
  | `background` | 背景颜色/图片 |
  | `width/height` | 尺寸 |
  | `display` | 布局类型 |
  | `position` | 定位方式 |
  | `overflow` | 溢出处理 |
  | `flex` / `grid` | 弹性布局/网格布局 |

# 强制继承和取消继承

- 强制继承：`inherit`
- 取消继承：`initial` 或 `unset`
