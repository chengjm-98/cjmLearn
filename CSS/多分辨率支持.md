# 媒体查询适配不同屏幕尺寸

```css
@media <media-type> and (<condition>) {
  /* CSS 样式 */
}
```

| 属性                        | 描述                                          |
| --------------------------- | --------------------------------------------- |
| `width` / `height`          | 视口宽高                                      |
| `min-width` / `max-width`   | 最小/最大宽度条件                             |
| `min-height` / `max-height` | 最小/最大高度条件                             |
| `orientation`               | 方向：`portrait`（竖屏）/ `landscape`（横屏） |
| `resolution`                | 分辨率（dpi / dppx）                          |

# 使用响应式单位 rem&flex

## rem 和 px 的区别

- px：绝对单位，固定大小，不随根字体变化
- rem：相对单位，基于 html 根元素的 font-size 计算

# 移动端适配

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```
