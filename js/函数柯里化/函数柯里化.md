# 函数柯里化

返回函数，链式调用
柯里化（Currying）是将一个接受多个参数的函数转化为一系列接受单一参数的函数，并且这些函数逐个返回一个接受下一个参数的函数。换句话说，柯里化将一个多参数函数转化为一系列嵌套的单参数函数。柯里化通常用于提高代码的可重用性和灵活性。

```jsx
//函数柯里化  实现add(1)(2)(3)() = 6
function add(a) {
  let sum = a;
  //或者可以写成
  function curry(b) {
    if (Array.from(arguments).length > 0) {
      sum += b;
      return curry;
    } else {
      //当b没有传值的时候就是合适的结束时间
      return sum;
    }
  }
  return curry;
}
console.log(add(1)(2)(3)());
```
