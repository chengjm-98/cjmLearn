# 提升

- 函数声明和变量声明都会提升，但是函数首先会被提升，然后才是变量。会将函数声明提升到作用域的最上面。
- var 声明的变量会被提升到作用域的最上面，会被赋值为 undefined，所以不会出现暂时性死区。
- let 和 const 声明的变量不会被提升到作用域的最上面，但是不会赋值。所以会有暂时性死区。
