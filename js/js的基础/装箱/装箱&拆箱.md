在 JavaScript 中，有两类类型：

- 原始类型（Primitive）：number、string、boolean、bigint、symbol、undefined、null
- 对象类型（Object）：所有非原始类型都是对象

- 原始类型本身没有方法和属性，但 JS 很方便地允许你在原始类型上使用方法。这就是装箱的作用：

# 什么是装箱/拆箱

- 装箱（Boxing）：当你访问原始值的属性或方法时，JS 会临时把它“包装”成对应的对象类型。
- 装箱：原始值 → 临时对象
- 用途：让原始类型可以访问方法和属性

- 拆箱：
- 对象 → 原始值
- 用途：取回原始值，进行数值运算或比较
