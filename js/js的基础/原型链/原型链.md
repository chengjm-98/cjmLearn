# 什么是原型

JavaScript 中的对象有一个特殊的[[Prototype]]内置属性,指向另外一个对象，这个对象就是它的原型。

# 创建自己的原型

Object.create 它会创建一个对象并把这个对象的[[Prototype]]关联到指定的对象。

```jsx
var anotherObject = {
  city: "Beijing",
};
//创建一个关联到anotherObject的对象
var myObject = Object.create(anotherObject);
console.log("是否正确", myObject.__proto__ == anotherObject);
```

anotherObject 是 myObject 的原型

# **_proto_** 和 prototype：

- 1.**_proto_** 和 Object.getPrototypeOf()：它们是**实例对象**的属性或方法，指向该对象的原型。
  - Object.getPrototypeOf() 是 ES5 引入的方法，用于获取对象的原型。
- 2.prototype：它是构造函数的属性，用来指向该构造函数创建的所有实例的原型。

# 原型链

当你访问对象的属性时，如果该对象本身没有这个属性，JavaScript 引擎就会查找它的原型，直到找到这个属性或达到原型链的顶端（即 null）。

- 但是到哪里是[[Prototype]]的“尽头”呢？
- 所有普通的[[Prototype]]链最终都会指向内置的 Object.prototype。
- Object.prototype 的原型指向 null
