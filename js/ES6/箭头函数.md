# 箭头函数

箭头函数没有自己的 this，它的 this 是在定义时确定的，而不是在调用时确定的。
箭头函数的 this 指向 外部函数的 this，而不是箭头函数的调用上下文。箭头函数没有自己的 this，它是 **词法绑定**的。

```js
const obj = {
  name: "Alice",
  greet() {
    setTimeout(() => {
      console.log(this.name);
      // `this` 指向外部的 `greet` 方法所在的 `obj`
    }, 1000);
  },
};
obj.greet(); // 输出 "Alice"
```
