# 箭头函数

箭头函数没有自己的 this，它的 this 是在**定义时确定**的，而不是在调用时确定的。
箭头函数的 this 指向 外部函数的 this，而不是箭头函数的调用上下文。箭头函数没有自己的 this，它是 **词法绑定**的。

```js
const obj = {
  name: "Alice",
  greet() {
    setTimeout(() => {
      console.log(this.name);
      // `this` 指向外部的 `greet` 方法所在的 `obj`
    }, 1000);
  },
};
obj.greet(); // 输出 "Alice"
```

# 箭头函数和普通函数的区别

## 箭头函数

- 箭头函数没有自己的 this，它的 this 是在**定义时确定**的，而不是在调用时确定的。箭头函数的 this 指向 外部函数的 this，而不是箭头函数的调用上下文。箭头函数没有自己的 this，它是 **词法绑定**的。
- 箭头函数没有 arguments 对象，它的 arguments 对象是在**定义时确定**的，而不是在调用时确定的。箭头函数的 arguments 对象是外部函数的 arguments 对象，而不是箭头函数的 arguments 对象。箭头函数没有自己的 arguments 对象，它是 **词法绑定**的。
- 箭头函数不能用作构造函数，不能使用 new 关键字来调用箭头函数。

## 普通函数

- 普通函数有自己的 this，它的 this 是在**调用时确定**的，而不是在定义时确定的。普通函数的 this 指向 调用它的对象，而不是它所在的上下文。普通函数的 this 是 **动态绑定**的。
- 普通函数有自己的 arguments 对象，它的 arguments 对象是在**调用时确定**的，而不是在定义时确定的。普通函数的 arguments 对象是调用它的 arguments 对象，而不是它所在的上下文。普通函数的 arguments 对象是 **动态绑定**的。
- 普通函数可以用作构造函数，使用 new 关键字来调用普通函数。
