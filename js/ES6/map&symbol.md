# map

- Map 是一种键值对集合，和 Object 类似，但键的类型不受限制。任何类型都可以作为 key（对象、函数、NaN 都可以）
- Map 本质是“哈希表”
- 顺序性：Map 中的键值对是按照插入顺序排序的
- 键不能重复：Map 中的键必须是唯一的，重复的键会被覆盖

## api

map.set(key, value)
map.get(key)
map.has(key)
map.delete(key)
map.clear()
map.size

# symbol

- Symbol 是一种“唯一且不可变”的原始数据类型，用于创建唯一的标识符。
- 解决 对象属性名冲突问题

# set

Set 是 ES6 引入的一种“值的集合”数据结构，用来存储不重复的值。

- 对象是按引用比较

# 区别

| 类型        | 是否可迭代 | key / value 要求   | 是否弱引用 | 是否可遍历 | 主要用途            |
| ----------- | ---------- | ------------------ | ---------- | ---------- | ------------------- |
| **Map**     | ✅         | key 任意类型       | ❌         | ✅         | 通用键值对集合      |
| **WeakMap** | ❌         | key 必须是对象     | ✅         | ❌         | 对象私有数据 / 缓存 |
| **Set**     | ✅         | value 任意类型     | ❌         | ✅         | 去重集合            |
| **WeakSet** | ❌         | value 必须是对象   | ✅         | ❌         | 对象状态标记        |
| **Symbol**  | ❌         | 唯一值（原始类型） | ❌         | ❌         | 唯一 key / 元编程   |

Map 和 Set 是可遍历、强引用的数据结构；
WeakMap 和 WeakSet 是不可遍历、弱引用结构，主要用于对象私有数据和防止内存泄漏；
Symbol 是唯一且不可变的原始数据类型，主要用于对象属性名冲突问题。
