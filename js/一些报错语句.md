# 一些用于判错的语句

## try-catch

- try-catch 捕获不到异步错误

## throw

- 抛出错误

  - 可以在 try-catch 中使用

  ```jsx
  try {
    throw new Error("错误");
  } catch (e) {
    //这里可以捕获到
    console.log(e);
  }
  但是在try - catch中的异步是无法捕获到的;
  try {
    setTimeout(() => {
      throw new Error("error"); // ❌ 不会被 try/catch 捕获
    }, 100);
  } catch (err) {
    console.log("不会执行");
  }
  ```

  - 也可以在 async 函数中使用
  - 也可以在 promise 中使用
