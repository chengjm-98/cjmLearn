# pop

- 从数组的末尾移除一个元素，并返回该元素。如果数组为空，则返回 undefined。
- 注意：pop 方法会改变原数组。
- 语法：array.pop()
- 参数：无
- 返回值：移除的元素
- 示例：
  ```js
  const arr = [1, 2, 3];
  arr.pop(); // 3
  console.log(arr); // [1, 2]
  ```

# shift

- 从数组的开头移除一个元素，并返回该元素。如果数组为空，则返回 undefined。
- 注意：shift 方法会改变原数组。
- 语法：array.shift()
- 参数：无
- 返回值：移除的元素
- 示例：
  ```js
  const arr = [1, 2, 3];
  arr.shift(); // 1
  console.log(arr); // [2, 3]
  ```

# splice

- 从数组中移除一个或多个元素，并返回被移除的元素。
- 注意：splice 方法会改变原数组。
- 语法：array.splice(start, deleteCount, item1, item2, ...)
- 参数：
  - start：要移除的元素的起始位置。
  - deleteCount：要移除的元素的数量。
  - item1, item2,...：要添加到数组中的元素。
- 返回值：被移除的元素组成的数组。
- 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.splice(1, 2); // [2, 3]
  console.log(arr); // [1, 4, 5]
  ```

# unshift

- 在数组的开头添加一个或多个元素，并返回新数组的长度。
- 注意：unshift 方法会改变原数组。
- 语法：array.unshift(item1, item2,...)
- 参数：
  - item1, item2,...：要添加到数组开头的元素。
- 返回值：新数组的长度。
- 示例：
  ```js
  const arr = [1, 2, 3];
  arr.unshift(0); // 4
  console.log(arr); // [0, 1, 2, 3]
  ```

# reverse

# sort

- 对数组中的元素进行排序。
- 注意：sort 方法会改变原数组。
- 语法：array.sort(compareFunction)
- 参数：
  - compareFunction：可选。用于指定排序顺序的函数。如果省略，则按照元素的 Unicode 码点顺序排序。
- 返回值：排序后的数组。
- 示例：
  ```js
  const arr = [3, 1, 2];
  arr.sort(); // [1, 2, 3]
  console.log(arr); // [1, 2, 3]
  ```

# concat

- 连接两个或多个数组，并返回一个新数组。
- 注意：concat 方法不会改变原数组。
- 语法：array.concat(value1, value2,...)
- 参数：
  - value1, value2,...：要连接到数组的元素。
- 返回值：连接后的新数组。
- 示例：
  ```js
  const arr1 = [1, 2];
  const arr2 = [3, 4];
  const arr3 = arr1.concat(arr2); // [1, 2, 3, 4]
  console.log(arr3); // [1, 2, 3, 4]
  ```

# join

- 将数组中的所有元素连接成一个字符串。
- 注意：join 方法不会改变原数组。
- 语法：array.join(separator)
- 参数：
  - separator：可选。指定分隔符。如果省略，则使用逗号作为分隔符。
- 返回值：连接后的字符串。
- 示例：
  ```js
  const arr = [1, 2, 3];
  arr.join(); // "1,2,3"
  console.log(arr); // [1, 2, 3]
  ```

# slice

- 返回一个新数组，包含从 start 到 end（不包括 end）的元素。
- 注意：slice 方法不会改变原数组。
- 语法：array.slice(start, end)
- 参数：
  - start：要提取的起始索引（包括该索引）。
  - end：要提取的结束索引（不包括该索引）。
- 返回值：提取的新数组。
- 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.slice(1, 3); // [2, 3]
  console.log(arr); // [1, 2, 3, 4, 5]
  ```

# indexOf

- 返回数组中第一个与指定值相等的元素的索引，如果不存在则返回 -1。
- 注意：indexOf 方法不会改变原数组。
- 语法：array.indexOf(searchElement, fromIndex)
- 参数：
  - searchElement：要查找的元素。
  - fromIndex：可选。开始查找的索引位置。
- 返回值：元素的索引，如果不存在则返回 -1。
- 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.indexOf(3); // 2
  console.log(arr); // [1, 2, 3, 4, 5]
  ```

# includes

- 判断数组是否包含指定的元素。
- 注意：includes 方法不会改变原数组。
- 语法：array.includes(searchElement, fromIndex)
- 参数：
  - searchElement：要查找的元素。
  - fromIndex：可选。开始查找的索引位置。
- 返回值：如果包含则返回 true，否则返回 false。

  ```js

  console.log(arr); // [1, 2, 3, 4, 5]
  ```

# find

- 返回数组中第一个满足测试函数的元素的值，如果没有找到则返回 undefined。
- 注意：find 方法不会改变原数组。
- 语法：array.find(callback(element, index, array), thisArg)
- 参数：
  - callback：用于测试每个元素的函数。
  - element：当前元素。
  - index：当前元素的索引。
  - array：调用 find 的数组。
  - thisArg：可选。执行 callback 时用作 this 的值。
- 返回值：第一个满足测试函数的元素的值，如果没有找到则返回 undefined。
- 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.find((element) => element > 3); // 4
  console.log(arr); // [1, 2, 3, 4, 5]
  ```

# map

- 返回一个新数组，数组中的元素为原始数组元素调用函数处理后的值。
- 注意：map 方法不会改变原数组。
- 语法：array.map(callback(currentValue, index, array), thisArg)
- 参数：
  - callback：用于处理每个元素的函数。
  - currentValue：当前元素。
  - index：当前元素的索引。
  - array：调用 map 的数组。
  - thisArg：可选。执行 callback 时用作 this 的值。
- 返回值：处理后的新数组。
  - 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.map((element) => element * 2); // [2, 4, 6, 8, 10]
  console.log(arr); // [1, 2, 3, 4, 5]
  ```

# forEach

- 对数组的每个元素执行一次提供的函数。
- 注意：forEach 方法不会改变原数组。
- 语法：array.forEach(callback(currentValue, index, array), thisArg)
- 参数：
  - callback：用于处理每个元素的函数。
  - currentValue：当前元素。
  - index：当前元素的索引。
  - array：调用 forEach 的数组。
  - thisArg：可选。执行 callback 时用作 this 的值。
- 返回值：无。
- 示例：
  ```js
  const arr = [1, 2, 3, 4, 5];
  arr.forEach((element) => console.log(element)); // 1 2 3 4 5
  console.log(arr); // [1, 2, 3, 4, 5]
  ```
