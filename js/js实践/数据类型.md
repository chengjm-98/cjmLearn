# 基础数据类型

Number：代表整数和浮动数字，例如 1、3.14、-100。

String：代表一串字符，例如 "hello"、'world'、`template literal`。

Boolean：代表真或假，值为 true 或 false。

Undefined：表示变量未赋值时的状态，默认值为 undefined。

Null：表示空值或无值，只有一个值 null。

Symbol：ES6 新增的原始数据类型，用于创建匿名且唯一的值。

BigInt：ES11 新增，表示大于 2^53 - 1 或小于 -(2^53 - 1) 的整数。

# 引用数据类型

- 对象（Object）：
  - 普通对象：由键值对组成的集合，例如 { key: value }。
  - 数组（Array）：有序的元素集合，例如 [1, 2, 3]。
  - 函数（Function）：可执行的代码块，例如 function() {}。
- 日期（Date）：表示特定的日期和时间，例如 new Date()。
- 正则表达式（RegExp）：用于匹配字符串的模式，例如 /pattern/。
- 错误（Error）：表示错误信息，例如 new Error("error message")。
- 其他对象：
  - 数学对象（Math）：包含数学运算的方法，例如 Math.sqrt(4)。
  - 全局对象（Global）：包含全局变量和函数，例如 window、document。
